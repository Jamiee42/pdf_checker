# PDF Checker 预检助手 使用指南 v1.0.0

## 1. 工具介绍
PDF Checker 预检助手是一款开源、免费的桌面工具，可用于识别 PDF 文件的页面类型，输出 Excel 格式的统计报告。

它易上手、自动化，能够帮你省去逐页翻看的时间，降低人工统计成本，提高工作效率。

### 1.1 谁在使用 PDF Checker？
PDF Checker 专为翻译行业从业人员开发，其分类规则服务于翻译排版实务需求。

v1.0.0 的核心能力：帮助用户**快速报价**、**预估工作量**、**合理分工**。

### 1.2 功能亮点
- **页面分析**：PDF Checker 能够读取 PDF 文件，将页面快速区分为**可编辑/不可编辑/空白**，也可识别文件异常状态（加密/损坏）并报错。
- **数据导出**：支持用户将结果保存为本地 Excel，便于存档、查看、编辑、分享。
- **置信度提示**：特殊页面（如转曲）标注复核建议，节省人工抽查精力。
- **GUI 友好**：界面简洁清晰，实时展现处理进度。
- **使用便利**：35MB 大小，即点即用，无需安装，无需修改系统设置。

[⬇️ 点击下载 Windows 版](https://github.com/Jamiee42/pdf_checker/releases/latest)

## 3. 快速开始
### 3.1 选择 PDF 文件
点击`浏览`，在弹出的文件选择窗口中选择 PDF 文件。

### 3.2 选择输出位置
点击`浏览`，选择 Excel 报告的保存位置和文件名。

> **说明**：默认名称为“[PDF 文件名]_Result.xlsx”。

### 3.3 开始分析
点击`开始分析`，程序将自动处理 PDF 文件。

> **说明**：下方进度条将实时展示处理进度，请等待处理完成，不要关闭程序窗口。

### 3.4 导出 Excel
分析完成后，点击`导出 Excel`，即可将统计报告保存至本地。

## 4. 结果解读
### 4.1 页面分类
| 页面类型 | 定义 | 处理建议 |
|------|------|------|
| 可编辑 | 包含可提取的文本层，支持直接选中、复制、搜索文本内容 | 可直接转换为可编辑格式（如通过`另存为`功能转为 Word、PPT 等）；转换后文本内容准确性无需额外核验 |
| 不可编辑 | 无文本层或文本层不可提取，通常为扫描图像或嵌入式图片，文本内容仅以图像形式存在，无法直接选中或搜索 | 需经 OCR 识别以提取文本，识别结果需人工核验；识别失败或精度不足的部分需手动转录 |
| 空白 | 无文本层、扫描图像或嵌入式图片等可视内容 | 无需处理，可根据项目需求保留或删除 |
| 无法识别 | 页面内容因加密或损坏等原因无法被识别 | 需要人工介入处理 |

### 4.2 Excel 「注意」列说明
| 提示类型 | 程序判断 | 说明 |
|------|------|------|
| 错误：文档加密 | 无法处理 | 工具不支持解密，请自行解密后重试 |
| 错误：文档损坏 | 无法处理 | 文件可能已损坏，请检查原文件 |
| 建议复核：疑似转曲 | 不可编辑 | 页面可能无法直接编辑 |
| 建议复核：仅含少量文字（疑似页眉页脚） | 空白页 | 页面可能只有页眉页脚，实际业务中可视为空白页 |
| 建议复核：无可识别字符，疑似转曲或乱码 | 不可编辑 | 页面可能无可用的正常字符 |
| 建议复核：含图像且文本较少 | 不可编辑 | 工具无法判断图片内容是否需要处理，按实务常见情况判定 |

> **说明**：标注「建议复核」的页面需要人工抽查，工具判断可能不准确。

### 4.3 当前版本的限制提示

以下情况**暂不会**出现在「注意」列中，需人工留意：

| 场景 | 程序判断 | 说明 |
|------|------|------|
| 双层 PDF（OCR 字符 ≥ 50） | 可编辑 | 上层是原始图像，下层是 OCR 识别文字，因其内容可能存在错误，实务中通常视为「**不可编辑**」 |
| 仅有表单域的页面 | 空白页 | 当前版本暂无表单域检测规则 |

> **说明**：以上场景将在后续版本中优化，届时将添加对应提示或规则。

## 5. 常见问题
**Q1：可以批量上传 PDF 吗？**

A：v1.0.0 阶段暂不支持，「批量上传」已列入功能计划。

**Q2：为什么我的文件分析特别慢？**

A：可能原因：
- 文件太大（建议单个文件 < 500 页或 < 50MB）
- 文件包含较多复杂字体或大量矢量图形

请耐心等待，如果超过 5 分钟仍无响应，关闭程序重试，或尝试拆分 PDF 后再分析。

**Q3：能处理加密文件吗？**

A：不能。工具不支持解密，仅检测并提示加密状态，请自行解密后重试。

**Q4：Excel 里为什么没有页码和占比？**

A：页码和占比列默认是隐藏的，目的是让核心信息更清晰。在 Excel 中选中相邻列，右键点击「取消隐藏」即可看到。

**Q5：工具判断和我的预期不一样？**

A：本工具为预检助手，旨在快速提供页面信息，不可取代人工判断。
如对结果有疑问，建议：
1. 查看 Excel「注意」列的复核提示
2. 对标注「建议复核」的页面进行人工抽检
3. 如确认是工具误判，欢迎反馈给开发者，帮助改进
